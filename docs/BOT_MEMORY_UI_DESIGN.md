## 页面元素说明

### 1. 页面头部
- **标题**: "机器人记忆"（左侧，大字体，深灰色）
- **返回按钮**: 右上角，浅灰背景，点击返回首页

### 2. 状态消息区
- **成功消息**: 绿色背景，深绿色文字，自动3秒后消失
- **错误消息**: 红色背景，深红色文字，需用户手动关闭

### 3. 加载状态
- **加载动画**: 旋转的圆圈动画
- **加载文字**: "加载中..."

### 4. 空状态
当没有机器人时显示：
```
    🤖
  还没有机器人
请先创建一个机器人来管理记忆
```

### 5. 机器人记忆卡片

#### 卡片头部
- **机器人头像**: 
  - 如果有头像URL: 显示48x48的圆角图片
  - 如果无头像: 显示渐变色圆角方块，内含首字母
- **机器人名称**: 粗体，1.25rem字体
- **机器人描述**: 次要文字，灰色，0.875rem字体
- **记忆状态徽章**:
  - "有记忆": 绿色背景，深绿色文字
  - "无记忆": 灰色背景，灰色文字

#### 记忆显示区（查看模式）
- **有记忆时**:
  - 浅灰背景的内容区域
  - 显示完整记忆文本（保持换行和格式）
  - 底部分隔线
  - 显示最后更新时间（小字，灰色）
  
- **无记忆时**:
  - 虚线边框的空状态区域
  - 居中显示"该机器人还没有记忆"

#### 记忆编辑区（编辑模式）
```
┌────────────────────────────────────────┐
│ 记忆内容                               │
│ ┌────────────────────────────────────┐ │
│ │ [多行文本输入框]                   │ │
│ │                                    │ │
│ │                                    │ │
│ │                                    │ │
│ └────────────────────────────────────┘ │
│ 记忆内容将帮助机器人更好地理解您的偏好和需求 │
└────────────────────────────────────────┘

[保存]  [取消]
```

#### 操作按钮
- **主要按钮（蓝色）**: 编辑记忆 / 添加记忆 / 保存
- **次要按钮（灰色）**: 从聊天生成 / 取消
- **危险按钮（红色）**: 删除记忆

### 6. 删除确认对话框

```
┌─────────────────────────────────────┐
│  确认删除                           │
├─────────────────────────────────────┤
│                                     │
│  确定要删除这个机器人的记忆吗？    │
│  此操作无法撤销。                  │
│                                     │
├─────────────────────────────────────┤
│              [取消]  [删除]         │
└─────────────────────────────────────┘
```
- 半透明黑色背景遮罩
- 白色圆角对话框居中显示
- 阴影效果增强层次感

## 颜色方案

### 主要颜色
- **主色调（蓝色）**: #3b82f6 (按钮、链接)
- **成功色（绿色）**: #10b981 (状态徽章、成功消息)
- **危险色（红色）**: #ef4444 (删除按钮、错误消息)
- **文字色**: 
  - 主要文字: #1f2937
  - 次要文字: #6b7280
  
### 背景颜色
- **页面背景**: 白色
- **卡片背景**: 白色 + 边框 #e5e7eb
- **内容区背景**: #f9fafb
- **按钮背景**: 
  - 主要: #3b82f6
  - 次要: #f3f4f6
  - 悬停: 深一档

### 渐变色（头像占位符）
- 渐变: 从 #667eea 到 #764ba2 (135度)

## 交互效果

### 悬停效果
- **卡片**: 增强阴影
- **按钮**: 背景色变深，平滑过渡
- **输入框**: 边框变蓝，外发光效果

### 动画
- **加载动画**: 旋转动画（1秒周期）
- **过渡动画**: 0.2秒平滑过渡（颜色、阴影、背景）
- **消息淡入淡出**: 3秒后自动消失

### 响应式设计
- **桌面端（>768px）**: 
  - 卡片宽度最大1200px，居中显示
  - 按钮横向排列
  
- **移动端（≤768px）**:
  - 卡片全宽显示
  - 按钮纵向堆叠，全宽显示
  - 头部标题和按钮响应式布局

## 用户体验细节

1. **即时反馈**: 所有操作都有加载状态和结果提示
2. **确认对话**: 删除操作需要二次确认
3. **自动保存提示**: 保存成功后3秒自动关闭提示
4. **错误处理**: 网络错误、空内容等都有友好提示
5. **禁用状态**: 保存中按钮禁用，防止重复提交
6. **空内容验证**: 不允许保存空记忆

## 可访问性

- 所有按钮都有描述性文本
- 适当的颜色对比度
- 键盘导航支持
- 响应式设计支持各种设备

## 技术实现

- **框架**: Angular 20 (Standalone Components)
- **样式**: SCSS with BEM-like naming
- **响应式**: Media queries + Flexbox
- **动画**: CSS transitions + animations
- **状态管理**: Component-level state with ChangeDetectorRef
